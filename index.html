What is this page?

This page is mostly intended to be a host for my cv, but I also wanted to play around with some css while I was at it. This page is completely javascript free, but is built using pug and sass for the sake of convenience.

The page is built to work well with mobile devices, and to output a print-friendly version of the cv in print mode. Just remember to set margins to none and background graphics to on in the print settings.
